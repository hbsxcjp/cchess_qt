
--------------------------------------------------------
                        前言

　　《三国七子谱》是目前所知最早的以七子棋局为体裁的棋局
谱，从时间上来说，要比付荣年的象棋七星谱还要早12年，它所
提的局名也很有特色，使棋局增添了不少趣味，着法短小精悍，
对排局排局爱好者有很大的参考价值。

　　吴怀洁乃是台湾排局家，于1939年起曾拟七子排局240多局,
在台湾报纸上连载过其中《三国七子谱》许多局，后除收入在由
李志海主编的《棋桥》五十局外，其余二百局左右，目前尚无法
收集，因棋桥是台湾出的棋谱，目前大陆稀见，殊为可贵，故排
局爱好者辗转传抄，视为珍品。

　　上海俞齐明棋友酷爱象棋，尤喜集谱，热心传播排局棋艺，
早在83年就计划刻印一些传世稀少的珍本孤本象棋排局谱，以广
为流传，继他翻刻付荣年的象棋残局新谱后，经过数月的工余努
力，现在《三国七子谱》已翻刻完成，又一本近代象棋排局谱奉
献给排局爱好者。
                   
                                 　　上海 蒋权

　　                               一九八四年五月
